과학고 정보 수업: 목표지향적 개념기반 수업지도안

수업 주제: III. 알고리즘과 프로그래밍 - 341. 표준 입출력: 프로그램과 세상의 첫 대화

수업 시작 멘트

심리학자 칼 융은 이렇게 말했습니다. "귀를 기울여 듣지 않는 자는, 내면의 목소리도 들을 수 없다." 프로그램도 마찬가지입니다. 사용자의 목소리(입력)를 듣지 못하는 프로그램은 외로운 독백에 불과하며, 자신의 생각(결과)을 세상에 들려주지(출력) 못하는 프로그램은 침묵하는 기계일 뿐입니다. 오늘 우리는 프로그램의 '귀'가 되는 input()과 '목소리'가 되는 print()를 배우며, 우리의 코드가 비로소 세상과 소통하고 상호작용하는 살아있는 존재가 되는 첫걸음을 떼겠습니다.

1. 학습 목표

개념적 목표

  • 프로그램이 외부 세계와 데이터를 주고받는 통로로서 '스트림(Stream)'의 개념을 이해하고, 표준 입력(stdin)과 표준 출력(stdout)이 각각 키보드와 모니터에 연결된 기본 스트림임을 설명할 수 있다.
  • input() 함수가 표준 입력 스트림으로부터 '문자열' 데이터를 읽어온다는 사실과, 이로 인해 숫자 연산을 위한 '타입 캐스팅'이 필수적임을 설명할 수 있다.

절차적 목표

  • input() 함수를 사용하여 사용자로부터 한 줄의 문자열 데이터를 입력받고 변수에 저장할 수 있다.
  • print() 함수를 사용하여 숫자, 문자열, 변수 등 다양한 데이터를 화면에 출력할 수 있다.
  • f-string과 sep, end 옵션을 활용하여 출력 형식을 원하는 대로 제어할 수 있다.

태도적 목표

  • 사용자에게 무엇을 입력해야 할지 명확히 안내하는 '프롬프트'의 중요성을 인지하고, 사용자 친화적인 프로그램을 작성하려는 태도를 갖는다.
  • TypeError나 ValueError가 발생했을 때, 입력 데이터의 타입 문제를 의심하고 논리적으로 디버깅하려는 자세를 갖는다.

2. 핵심 개념 상세 설명

스트림 (Stream)

프로그램과 외부 장치(키보드, 모니터, 파일 등) 사이에서 데이터가 오고 가는 단방향 통로를 의미합니다. 물이 한 방향으로 흐르는 시냇물과 같다고 해서 스트림이라고 부릅니다.

  • 입력 스트림 (Input Stream): 데이터가 외부 장치에서 프로그램으로 들어오는 통로.
  • 출력 스트림 (Output Stream): 데이터가 프로그램에서 외부 장치로 나가는 통로.

표준 입출력 (Standard I/O)

프로그램이 실행될 때, 별도의 지정 없이 운영체제가 자동으로 마련해주는 가장 기본적인 입출력 스트림입니다.

  • 표준 입력 (stdin, Standard Input): 기본적으로 '키보드'에 연결됩니다. 파이썬의 input() 함수가 이 통로를 통해 데이터를 읽어들입니다.
  • 표준 출력 (stdout, Standard Output): 기본적으로 '모니터(터미널 화면)'에 연결됩니다. 파이썬의 print() 함수가 이 통로로 데이터를 내보냅니다.
  • 표준 에러 (stderr, Standard Error): 오류 메시지를 출력하기 위한 별도의 통로로, 기본적으로 모니터에 연결됩니다.

함수 상세

  • input([prompt]) 함수 - 프로그램의 귀: 표준 입력(키보드)으로부터 사용자가 한 줄을 입력하고 엔터 키를 누를 때까지 기다렸다가, 입력된 내용을 '문자열(string)' 형태로 반환합니다.
    • prompt: 사용자에게 무엇을 입력해야 할지 알려주는 안내 메시지. (예: input("이름을 입력하세요: "))
    • 핵심: input()은 숫자를 입력해도 무조건 문자열로 반환하므로, 숫자 계산을 위해서는 int()나 float()으로 반드시 형 변환(Type Casting)을 해야 합니다.
  • print(*objects, sep=' ', end='\n') 함수 - 프로그램의 목소리: 주어진 객체들을 문자열로 변환하여 표준 출력(화면)에 내보냅니다.
    • *objects: 출력하고 싶은 여러 개의 값들을 쉼표로 구분하여 전달합니다.
    • sep: 'separator'의 약자로, 여러 개의 값을 출력할 때 그 사이를 무엇으로 구분할지 지정합니다. 기본값은 공백(' ')입니다.
    • end: 모든 값의 출력이 끝난 뒤, 마지막에 무엇을 붙일지 지정합니다. 기본값은 줄바꿈 문자('\n')입니다.

3. 수업의 흐름 상세 기술

1단계: 도입 (5분) - 소통의 시작

"위대한 과학자 갈릴레오 갈릴레이는 그의 저서 '두 우주 체계에 관한 대화'에서, 지동설을 주장하는 인물과 천동설을 주장하는 인물을 등장시켜 서로 '대화'하게 함으로써 자신의 과학적 사상을 펼쳤습니다. 프로그램도 마찬가지입니다. 일방적으로 자신의 이야기만 하는 프로그램은 살아있다고 느끼기 어렵습니다. 사용자와 '대화'할 수 있을 때, 즉 사용자의 말을 듣고(input) 자신의 생각을 말할(print) 수 있을 때 비로소 프로그램은 생명력을 얻습니다. 오늘은 이 소통의 기술을 배우겠습니다."

2단계: 전개 (35분) - 듣고 말하기

활동 1: 프로그램의 목소리 - print의 다양한 얼굴

print() 함수의 다양한 사용법을 단계별로 보여줍니다.

# 기본 출력
print("Hello, World!")
print(123)

# 여러 값 출력 (쉼표로 구분하면 자동으로 공백이 삽입됨)
print("원주율:", 3.14, "입니다.")
# 출력: 원주율: 3.14 입니다.

# sep 활용: CSV(Comma-Separated Values) 데이터 만들기
year = 2024
month = 2
day = 14
# sep=',' 옵션으로 각 값을 쉼표로 구분하여 출력
print(year, month, day, sep=',') # 출력: 2024,2,14
print("실험 데이터를 쉼표로 구분된 형식으로 저장할 때, sep 옵션을 사용하면 매우 편리합니다.")

# end 활용: 로딩 진행 상태 시뮬레이션
import time
print("데이터 처리 중... ", end="")
time.sleep(1) # 1초 대기
print("50%", end="")
time.sleep(1)
print(" 100% 완료.")
print("기본적으로 print는 항상 줄을 바꾸지만, end='' 옵션을 주면 줄바꿈 없이 다음 출력이 같은 줄에 이어지게 할 수 있습니다.")

활동 2: 프로그램의 귀 - input과 타입 캐스팅의 중요성

사용자로부터 두 숫자를 입력받아 더하는 프로그램을 만드는 과정을 통해 input()의 함정을 보여줍니다.

잘못된 시도:
num1_str = input("첫 번째 숫자: ") # 사용자가 '10' 입력
num2_str = input("두 번째 숫자: ") # 사용자가 '20' 입력
result = num1_str + num2_str
print("두 수의 합:", result) # 결과: "1020"
"왜 30이 아니라 1020이 나왔을까요?" type(num1_str)을 출력하여 'str'임을 보여주며, 문자열 덧셈은 '이어 붙이기'임을 상기시킵니다.
올바른 해결책:
num1_str = input("첫 번째 숫자: ")
num2_str = input("두 번째 숫자: ")

# 입력받은 '문자열'을 '숫자(정수)'로 형 변환
num1_int = int(num1_str)
num2_int = int(num2_str)

result = num1_int + num2_int
print("두 수의 합:", result) # 결과: 30
"사용자로부터 받은 모든 입력은 '일단 의심'하고, 우리가 원하는 '타입으로 변환'하는 과정이 필수적입니다. 이것이 바로 견고한 프로그램의 시작입니다." (자유 낙하 공식 s = v_{0}t + ½at^{2}의 변수들을 input과 float()으로 받아 계산하는 예제를 함께 풀어봅니다.)

3단계: 정리 (5분) - 소통의 파이프라인

'사용자(키보드) → input() → 프로그램(문자열 처리 및 계산) → print() → 사용자(모니터)'로 이어지는 데이터의 흐름, 즉 표준 입출력 파이프라인을 그림으로 그려주며 정리합니다. "input()은 항상 문자열을 반환하고, print()는 무엇이든 출력한다. 그리고 그 사이에서 데이터를 올바른 형태로 '가공(형 변환)'하는 것이 프로그래머의 역할"임을 강조하며 마무리합니다.

수업 마무리 멘트

철학자 마르틴 부버는 '나와 너'라는 책에서, 진정한 관계는 상대를 사물(It)로 대하는 것이 아니라 인격(You)으로 대하는 '만남'에서 시작된다고 말했습니다. 오늘 우리는 프로그램을 단순히 명령을 실행하는 기계(It)가 아니라, 사용자와 대화하고 상호작용하는 파트너(You)로 만드는 법을 배웠습니다. 여러분이 만든 프로그램이 사용자의 말에 귀 기울이고, 친절한 목소리로 답하며 세상과 진정한 '만남'을 갖게 되기를 바랍니다.

4. 진단 및 형성평가

진단평가 (OX 퀴즈 10개)

1. input() 함수로 숫자 100을 입력하면, 반환되는 값의 자료형은 int이다. (X)

풀이: input() 함수는 사용자가 키보드로 입력한 모든 것을 예외 없이 '문자열(str)'로 반환합니다. 따라서 숫자 100을 입력해도 그 결과는 정수 100이 아닌 문자열 '100'입니다.

2. print(10, 20) 코드는 화면에 "1020"을 출력한다. (X)

풀이: print() 함수에서 쉼표(,)로 여러 값을 구분하면, 각 값 사이에 기본적으로 공백 한 칸이 자동으로 삽입됩니다. 따라서 "10 20"이라고 출력됩니다.

3. age = int(input("나이: ")) 코드는 사용자 입력을 받아 정수로 변환 후 age 변수에 저장하는 올바른 한 줄 표현이다. (O)

풀이: 이 코드는 함수의 중첩을 잘 활용한 예입니다. 안쪽의 input() 함수가 먼저 실행되어 사용자로부터 문자열을 입력받고, 그 반환된 문자열이 바로 바깥쪽의 int() 함수에 전달되어 정수로 변환됩니다.

4. print("Hello", end="") 코드는 "Hello"를 출력한 뒤 줄을 바꾸지 않는다. (O)

풀이: print() 함수의 end 인자는 출력의 마지막에 붙일 문자를 지정하며, 기본값은 줄바꿈을 의미하는 '\n'입니다. end=''로 지정하면 줄바꿈 문자 대신 빈 문자열을 붙이므로, 다음 출력이 바로 같은 줄에 이어지게 됩니다.

5. input() 함수의 괄호 안에는 아무것도 넣을 수 없다. (X)

풀이: input() 함수의 괄호 안에는 사용자에게 보여줄 안내 메시지, 즉 '프롬프트(prompt)' 문자열을 넣을 수 있습니다. 예: input("이름을 입력하세요: ")

6. 표준 입력(stdin)은 기본적으로 모니터에, 표준 출력(stdout)은 키보드에 연결된다. (X)

풀이: 정반대입니다. 표준 입력(stdin)은 키보드, 표준 출력(stdout)은 모니터(터미널 화면)에 기본적으로 연결됩니다.

7. print("A", "B", "C", sep="-")의 결과는 "A-B-C"이다. (O)

풀이: print() 함수의 sep 인자는 'separator'(구분자)의 약자로, 출력할 값들 사이에 들어갈 문자를 지정합니다. 기본값인 공백(' ') 대신 하이픈('-')을 지정했으므로, 각 문자 사이에 하이픈이 들어가 "A-B-C"가 출력됩니다.

8. f-string은 변수와 문자열을 조합하여 출력하는 데 사용되는 편리한 기능이다. (O)

풀이: name = "홍길동"; print(f"안녕, {name}") 처럼, 문자열 앞에 f를 붙이고 중괄호 {} 안에 변수나 표현식을 넣어 간단하게 문자열을 형식화할 수 있습니다.

9. int("3.14") 코드는 소수점 이하를 버리고 정수 3으로 변환된다. (X)

풀이: int() 함수는 정수 형태의 문자열만 변환할 수 있습니다. '3.14'처럼 소수점이 포함된 문자열을 int()로 변환하려고 시도하면 ValueError가 발생합니다.

10. print() 함수의 sep과 end는 반드시 함께 사용해야 한다. (X)

풀이: sep과 end는 선택적인 키워드 인자(optional keyword argument)입니다. 필요에 따라 둘 중 하나만 사용하거나, 둘 다 사용하거나, 둘 다 사용하지 않을 수 있습니다.

형성평가 (5지선다형 10개)

1. 다음 input() 함수에 대한 설명 중 가장 정확한 것은?

  1. 사용자로부터 숫자를 입력받아 정수(int) 타입으로 반환한다.
  2. 사용자로부터 입력을 받되, 입력값의 데이터 타입을 스스로 분석하여 반환한다.
  3. 사용자로부터 입력받은 모든 것을 항상 문자열(str) 타입으로 반환한다.
  4. 괄호 안에 반드시 출력할 안내 메시지를 포함해야 한다.
  5. print() 함수와 동일한 기능을 수행한다.
정답 및 해설: ③ (input() 함수의 가장 중요하고 핵심적인 특징은 사용자가 숫자, 문자, 기호 등 무엇을 입력하든 그 결과를 하나의 '문자열' 덩어리로 취급하여 반환한다는 점입니다. 이로 인해 숫자 연산을 위해서는 반드시 타입 변환이 필요합니다.)

2. 아래 코드의 실행 결과로 올바른 것은? (사용자가 10과 20을 입력했다고 가정)

x = input()
y = input()
print(x + y)
  1. 30
  2. 1020
  3. "1020"
  4. 10 20
  5. TypeError 발생
정답 및 해설: ② (input()은 문자열을 반환하므로 x는 '10', y는 '20'이 됩니다. 문자열에 + 연산자를 사용하면 두 문자열을 이어 붙이는 '연결(concatenation)'이 일어나므로, '1020'이라는 새로운 문자열이 생성되고 이것이 화면에 출력됩니다. 화면에는 따옴표 없이 1020이 보입니다.)

3. 사용자로부터 반지름을 입력받아 원의 넓이를 계산하려고 할 때, 다음 중 가장 올바른 코드 조각은? (pi는 3.14로 가정)

  1. r = input("반지름: "); area = 3.14 * r**2
  2. r = int(input("반지름: ")); area = 3.14 * r**2

상세한 풀이와 설명: print 함수의 end 옵션을 응용한 문제입니다. end 값을 공백 세 칸(' ')으로 지정하면, print 함수는 단어를 출력한 뒤 줄을 바꾸는 대신 공백 세 칸을 출력합니다. 마지막 print 함수에서는 end를 지정하지 않았으므로, 기본값인 줄바꿈이 적용되어 출력이 마무리됩니다. 이 방법은 sep을 사용하는 것과는 다른 접근으로 출력 형식을 제어하는 방법을 보여줍니다.
